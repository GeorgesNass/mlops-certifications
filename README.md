# MLOps Certifications and Projects

This repository contains hands-on projects and practical exercises completed as part of an **MLOps training path**, aligned with industry practices and certification-oriented curricula such as the DataScientest MLOps course.

It showcases concrete implementations of core MLOps concepts through real code and tooling.
